# NestJS + React Application

## Usage

## Prerequisites

```
- Docker
```

### Environment Variables

```dotenv
# db config
POSTGRES_USER=
POSTGRES_PASSWORD=
POSTGRES_DB=
```

### Offline Mode

Backend is available in `3000` port, and Frontend in `5173`

```
# Running services
docker-compose up
```

## Documentation

Endpoints docs in swagger are available in `/docs` path of the backend 

```
> Client endpoints

  - Post client
  - Patch client

```
